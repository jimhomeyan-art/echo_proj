   - 当前筹码峰分析（形成日期、距今交易日数、价格点位）

## 筹码分析理论

### 筹码分布原理

筹码分布反映了在不同价位上投资者的持仓成本分布情况：
- **底部筹码**：低位密集区，代表长期持有者、机构建仓区
- **顶部筹码**：高位密集区，代表短期获利者、追高买入区
- **筹码峰**：成交最密集的区域，即最长筹码所在位置

### 筹码峰时间含义

- **近期形成的筹码峰**：说明短期交易活跃，可能是短线资金进场
- **远期形成的筹码峰**：说明长期横盘整理，筹码高度锁定
- **筹码峰前移**：筹码从远期向近期转移，说明主力在派发
- **筹码峰后移**：筹码从近期向远期转移，说明主力在吸筹

### 核心信号判断规则

| 信号类型 | 底部筹码变化 | 顶部筹码变化 | 操作建议 |
|----------|-------------|-------------|----------|
| 🔴 撤退信号 | 缩短（减少） | 变长（增加） | 减仓或离场 |
| 🟢 加仓信号 | 变长（增加） | 缩短（减少） | 逢低加仓 |
| 🟡 观望信号 | 无明显变化 | 无明显变化 | 保持观望 |

#### 🔴 撤退信号详解
**条件：底部最长筹码缩短 + 顶部筹码变长**

具体表现：
- 底部区域的最长筹码（远期形成的）开始缩短
- 顶部区域（近期形成）的筹码增加
- 筹码峰从远期向近期转移
- 解读：主力资金在高位派发，短线资金在追高
- **操作建议：减仓或离场**

#### 🟢 加仓信号详解
**条件：底部最长筹码变长 + 顶部筹码缩短**

具体表现：
- 底部区域的最长筹码（远期形成的）开始增长
- 顶部区域（近期形成）的筹码减少
- 筹码峰从近期向远期转移
- 解读：主力资金在低位吸筹，锁定低位筹码
- **操作建议：逢低加仓**

#### 🟡 观望信号详解
**条件：筹码分布无明显变化**

具体表现：
- 底部最长筹码和顶部筹码都保持稳定
- 筹码峰位置无明显移动
- 解读：市场处于震荡整理阶段，多空力量平衡
- **操作建议：保持观望，等待方向明确**

import pandas as pd

# 方法1：使用akshare获取
def get_sh_chip_data():
    try:
        # 获取上证指数的筹码分布数据
        df = ak.stock_china_em_ywbm(symbol="sh000001")
        return df
    except Exception as e:
        print(f"获取数据失败: {e}")
        return None

def get_sh_cost_distribution():
    try:

        return df
    except Exception as e:
        print(f"获取数据失败: {e}")
        return None
