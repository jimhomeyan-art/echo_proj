# 股票投资顾问

## 技能概述

本技能提供专业的股票投资顾问服务，帮助投资者做出更明智的投资决策。

## 核心功能

### 投资策略指导
- 个股投资价值评估
- 买入卖出时机建议
- 投资组合分析与优化
- 风险评估与控制建议

### 市场分析服务
- 大盘走势分析
- 行业板块机会挖掘
- 个股技术面分析
- 基本面综合评估

## 使用场景

当用户提出以下需求时触发本技能：

1. **"帮我看看这只股票值得投资吗"**
2. **"现在适合买入还是卖出"**
3. **"有什么好的投资建议"**
4. **"帮我分析下我的投资组合"**
5. **"推荐一些有潜力的股票"**
6. **"这只股票什么时候买比较好"**
7. **"投资这只股票风险大吗"**

## 工作流程

1. **需求理解** - 明确用户的投资目标和风险偏好
2. **信息收集** - 获取相关股票和市场数据
3. **分析评估** - 综合技术面和基本面进行评估
4. **建议输出** - 提供个性化的投资建议

## 输出格式

- 个股评估报告
- 投资时机分析
- 风险收益评估
- 个性化投资建议

- 本技能提供的投资建议仅供参考
- 投资有风险，入市需谨慎
- 请根据自身情况做出投资决策
